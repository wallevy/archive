* [Game Rom](./rom/)

An [NES](https://en.wikipedia.org/wiki/Nintendo_Entertainment_System) file is a game ROM created from an NES (Nintendo Entertainment System) video game. It contains the same data as the original NES cartridge and can be opened and played on a Mac or PC using an NES emulator.

* [OpenEmu](https://github.com/OpenEmu/OpenEmu)

OpenEmu is an open source project whose purpose is to bring macOS game emulation into the realm of first class citizenship.
